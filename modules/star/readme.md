# Star

Star shape.


## Installation

    npm install @pencil.js/star


## Examples

```js
import Star from "@pencil.js/star";

const radius = 50;
const branches = 6;
const bevel = 0.7;
const options = {
    fill: "gold",
    stroke: "goldendrod"
};
let star = new Star(aPosition, radius, branches, bevel, options);
```
